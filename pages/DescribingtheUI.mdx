# Describing the UI

## চলো বুঝি UI কিভাবে তৈরি হয়

React একটি জাভাস্ক্রিপ্ট লাইব্রেরি, যেটা দিয়ে খুব সহজে UI বানানো যায়। যেটা আমরা স্ক্রিনে দেখি তাই UI। ছোট ছোট উপাদান দিয়ে UI বানানো হয়— যেমন বাটন, লেখা, ছবি। React দিয়ে আমরা এইসব উপাদান কমবাইন করে একটি কম্পোনেন্ট তৈরি করি, যা নেস্ট করে করে সম্পূর্ণ UI বানানো যায়। এদেরকে প্রয়োজনে বার বার UI তে ব্যবহার করা যায়। কম্পোনেন্ট হলো— একটি বড় এপ্লিকেশনকে ভেঙ্গে টুকরো টুকরো করা, যার দায়িত্ব একটি সিংগেল ডিউটি পালন করা। এই অধ্যায়ে আমরা শিখব, কিভাবে React কম্পোনেন্ট বানানো যায়, নিজের মতো ডিজাইন করা যায় এবং দরকার হইলে দেখানো বা না দেখানোর কৌশল শিখবা।
